# Minty
**Minty Fresh icon pack for the Cinnamon Desktop**

![Image](https://github.com/tnetenbaa/minty/blob/main/preview.png)

This project is a fork of the Mint-Y icon theme, with custom folders and icons that help bring a modern yet familiar feel.

## Installation Instructions

**For User Install:** copy to ~/.icons or ~/.local/share/icons

**For System Install:** copy to /usr/share/icons
